# NOVA Cuisine — configurateur & rendu IA

Un configurateur de cuisine sur mesure : on choisit l'implantation, l'îlot, les finitions,
l'électroménager, on importe une photo de la pièce, et le site génère un rendu
photoréaliste de la future cuisine dans cette pièce (via l'API Replicate, modèle
`black-forest-labs/flux-kontext-pro`).

## Fichiers à conserver à la racine
- `index.html`
- `package.json`
- `api/render.js`
- `api/prediction.js`
- `api/diagnostic.js`

Ne créez pas de `server.js`, `serveur.js` ou de fichier `vercel.json` : le site est
100 % statique + fonctions serverless, Vercel les détecte automatiquement.

## Mise en ligne (Vercel, gratuit) — pas à pas

### 1. Créer un jeton Replicate
1. Allez sur https://replicate.com et créez un compte.
2. Renseignez un moyen de paiement dans les paramètres du compte (Replicate facture
   à l'usage chaque génération d'image, quelques centimes par rendu).
3. Ouvrez https://replicate.com/account/api-tokens et copiez votre jeton (il commence
   par `r8_...`). Gardez-le de côté, vous en aurez besoin à l'étape 3.

### 2. Connecter le dépôt GitHub à Vercel
1. Allez sur https://vercel.com et cliquez sur **Sign Up**.
2. Choisissez **Continue with GitHub** et autorisez l'accès à votre compte GitHub.
3. Une fois connecté, cliquez sur **Add New… → Project**.
4. Sélectionnez le dépôt `nova-cuisine` dans la liste (autorisez Vercel à accéder au
   dépôt si demandé).

### 3. Configurer la variable d'environnement
Avant de cliquer sur Deploy, dans l'écran de configuration du projet :
1. Ouvrez la section **Environment Variables**.
2. Nom : `REPLICATE_API_TOKEN`
3. Valeur : le jeton copié à l'étape 1.
4. Cliquez sur **Add**, puis sur **Deploy**.

Aucun autre réglage n'est nécessaire (pas de build command, pas de framework à choisir).

### 4. Vérifier le déploiement
Une fois le déploiement terminé, Vercel donne une URL du type
`https://nova-cuisine-xxxx.vercel.app`. Ouvrez :

`https://VOTRE-SITE.vercel.app/api/diagnostic`

Le résultat doit indiquer `"ok": true`. Si ce n'est pas le cas, le message d'erreur
indique généralement si le jeton est manquant, invalide, ou si le compte Replicate
n'a pas de moyen de paiement actif.

### 5. Tester le site
Ouvrez `https://VOTRE-SITE.vercel.app`, configurez une cuisine, importez une photo
de pièce (idéalement 1000 px de large ou plus, bien éclairée) et cliquez sur
**Générer le rendu dans ma pièce**. Le premier rendu peut prendre jusqu'à quelques
minutes.
