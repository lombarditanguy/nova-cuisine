# NOVA Cuisine — site test

## Déploiement Vercel
1. Importez ce dossier complet dans un dépôt GitHub.
2. Importez le dépôt dans Vercel.
3. Dans **Settings > Environment Variables**, ajoutez :
   - `REPLICATE_API_TOKEN` : votre jeton secret.
   - `REPLICATE_MODEL` (optionnel) : `black-forest-labs/flux-kontext-pro`.
4. Redéployez le projet.

Le jeton ne doit jamais être ajouté dans le code ni dans GitHub.
