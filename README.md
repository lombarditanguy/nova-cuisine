# NOVA Cuisine — version propre

Ce dossier est prêt pour un déploiement avec :
- `index.html` à la racine
- `api/render.js` à la racine du dossier `api`
- `package.json` à la racine

Important : ne pas ajouter `server.js` et ne pas ajouter `vercel.json`.
Dans les variables d'environnement du projet, ajouter :
`REPLICATE_API_TOKEN` = votre jeton de rendu.

Ensuite, envoyez le contenu de ce dossier à la racine du dépôt puis validez les changements.
